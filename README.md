## 🌱 Why Lirio

Reading real books is one of the most powerful ways to acquire a language. The problem? Real books are usually **too hard, too soon** — so learners give up before the magic happens.

**Lirio fixes that.** Every book is rewritten and summarized at *graded difficulty levels*, so you start where you are and climb naturally. You're never lost, never bored, and always reading something worth reading.

> 💡 **The promise:** read your way from your first French sentence to your last great novel — and arrive fluent.

---

## 🪜 The Ladder

Grasp a book at one level, climb to the next. Each rung is fully self-contained.

| Level | CEFR | What it feels like |
|:-----:|:----:|:-------------------|
| 🟢 | **A1** | Short sentences, core words, lots of support |
| 🟡 | **A2** | Everyday language, simple connected ideas |
| 🟠 | **B1** | Real arguments and storylines, lighter scaffolding |
| 🔴 | **B2** | Nuance, abstraction, near-original richness |
| ⚫ | **Native** | The book, unfiltered |

```
🟢 A1  →  🟡 A2  →  🟠 B1  →  🔴 B2  →  ⚫ Native
```

## 🧩 Inside Every Chapter

Every chapter of every book follows the same dependable structure:

| Section | What you get |
|:--|:--|
| 📖 **Summary** | The chapter's key ideas, written at your level |
| 🔤 **Vocabulary** | Meaning first, with context only when it truly helps |
| 💬 **Expressions & Idioms** | The natural phrases native speakers actually use |
| 🧠 **Grammar** | One focused grammar point, illustrated in context |
| ✍️ **Exercises** | Practice that locks in what you just read |
| 📝 **Word Glosses** | Kindle-style tap-to-define explanations |
| 🔊 **Audio** | Text-to-speech recordings to train your ear |

## 📂 Repository Structure

The language lives at the top level, so adding new languages later stays clean.

```
lirio/
├── french/
│   ├── self-development/
│   │   └── 7-habits-of-highly-effective-people/
│   │       ├── book.md            # overview + reading order
│   │       ├── a1/
│   │       │   ├── chapter-01.md
│   │       │   └── chapter-01.mp3
│   │       ├── a2/
│   │       └── b1/
│   └── novels/
│       └── le-petit-prince/
│           └── a1/
│               ├── chapter-01.md
│               └── chapter-01.mp3
├── templates/
│   └── chapter-template.md         # keeps all 200 books consistent
└── README.md
```

> 🌍 Future languages simply become new top-level folders: `spanish/`, `german/`, `italian/` …
